Inspetor e Elementos

Na aula de Inspetor e Elementos foi interessante conhecer como podemos visualizar as tags que foram escritas dentro de um site a partir da opção Ferramentas do Programador. Ferramentas que nos permite aprender sobre HMTL lendo estruturas HMTL prontas, no caso, desenvolvida por outro programador.

Estrutura básica do HTML

Quando falamos sobre HTML, é importante conhecer sobre a estrutura básica do HMTL, e é exatamente o que consegui aprender neste módulo. As tags head, title, html, body, que nos dão opções para criar o cabeçalho do site, colocar um título e desenvolver a interface que o usuário vai visualizar.

Falando sobre Tags

Ainda continuando falando sobre Tags, podemos destacar o uso das tags i e strong que são usadas para deixar textos e palavras em itálico ou em negrito, podendo variar o aparência do site ou destacar palavras e textos específicos

Atributos das Tags

Entre os atributos abordados na aula, podemos destacar os atributos das tags Input e IMG, sendo respectivamente cada uma delas, os atributos "type" e "src", amplamente usados para inserir números, textos, imagens, emojis e muito mais.

Textos

Sem dúvidas, os textos dentro das páginas HTML são uma das coisas que mais podemos variar ao longo da estrutura de uma página HTML. A inserção de palavras dentro de "strong", "mark", "blockquote", "sup", ensinadas na aula são de extrema importância pro desenvolvimento dos nossos sites, com eles, aprendi que podemos definir a posição de textos, colocar os mesmos em negrito, adicionar asteriscos ou sublinhar palavras.

Listas Ordenadas e não ordenadas

Aprender sobre o conceito de Listas Ordernadas foi super didático. Neste módulo, aprendi com facilidade a diferença entre "ol" e "ul", como adcionar listas dentro dessas tags com "li" e organizar possívei itens dentro uma página.

Links

A aula sobre links foi de extrema importância. O modo como podemos usar tags como "" para incluir links em HTML foi muito fácil de compreender. A tag target que permite o usuário acessar um link dentro da mesma página ou em outra página foi interessante de aprender. O atributo "title="Clque aqui e confira"" foi um recurso que amei conhecer. E sem falar, claro, do "about.html" que podemos incluir dentro de links para adicionar mais informações sobre o conteúdo da página HTML que está sendo desenvolvia.
